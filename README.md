# Repo de test Claude Code

Ce dépôt est utilisé pour tester **Claude Code** — l'assistant IA en ligne de commande d'Anthropic.

## À propos

[Claude Code](<https://claude.ai/code>) permet d'interagir avec un agent IA directement depuis le terminal ou via l'interface web, pour écrire du code, corriger des bugs, créer des fonctionnalités, et plus encore.
